# Zaman İstihbaratı Satış Dashboard’u (Power BI)

Bu proje, Power BI kullanarak gelişmiş **Zaman İstihbaratı analizi** göstermektedir.  
Satış performansını yıllık, ay bazlı ve yıllar arası karşılaştırmalı olarak analiz etmeye odaklanır. Ayrıca 12 aylık rolling trend ve 3 aylık hareketli ortalamalar içerir.

## Özellikler

- **Özel Tarih Tablosu**: Yıl, Ay, Çeyrek ve Yıl-Ay kolonlarıyla tüm tarihler kapsanır.
- **Temel Measure’lar**:
  - Toplam Satış
  - YTD Satış (Yıl Başından Bugüne)
  - MTD Satış (Ay Başından Bugüne)
  - Geçen Yıl Satış
  - YoY Büyüme %
  - YTD Büyüme %
  - 12 Aylık Rolling Satış
  - 3 Aylık Hareketli Ortalama
- **Görselleştirmeler**:
  - KPI Kartları ile hızlı bilgi
  - Line Chart: Toplam Satış, Rolling 12 Aylık Satış, Hareketli Ortalama
  - Slicer: Yıl ve Ay seçimi
- **Profesyonel Tasarım**:
  - KPI’lar için Conditional Formatting
  - Temiz ve executive-friendly layout


# Time Intelligence Sales Dashboard (Power BI)

This project demonstrates advanced **Time Intelligence analysis** using Power BI.  
It focuses on analyzing sales performance over time, including Year-to-Date (YTD), Month-to-Date (MTD), and Year-over-Year (YoY) growth, as well as rolling 12-month trends.

## Features

- **Custom Date Dimension**: A full date table covering all relevant dates with Year, Month, Quarter, and Year-Month columns.
- **Key Measures**:
  - Total Sales
  - YTD Sales
  - MTD Sales
  - Last Year Sales
  - YoY Growth %
  - YTD Growth %
  - Rolling 12-Month Sales
  - 3-Month Moving Average
- **Visualizations**:
  - KPI Cards for quick insights
  - Line Charts showing total sales, rolling 12-month sales, and moving averages
  - Slicers for dynamic year and month selection
- **Professional Design**:
  - Conditional formatting for KPIs
  - Clean, executive-friendly layout

## Purpose

- Demonstrates ability to model data in **Power BI** with star schema design
- Implements **time intelligence calculations** for trend analysis
- Creates a **dashboard suitable for executive decision-making**

## Skills Highlighted

- DAX (Time Intelligence functions: TOTALYTD, TOTALMTD, DATEADD, SAMEPERIODLASTYEAR, etc.)
- Data modeling and relationships in Power BI
- Advanced visualization techniques
- Rolling averages and trend analysis
